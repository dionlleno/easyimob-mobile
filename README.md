# EASYIMOB - MOBILE

Projeto Extensionista para a disciplina "Desenvolvimento Mobile"

**Objetivo:** Criar um aplicação mobile usando React Native e Expo Go

**Foco:** Um gerenciador capaz de armazenar dados dos clientes, dos imoveis e os compromissos, para corretores imobiliarios.

**Funções:**
* Manter cadastro de Clientes e Imoveis
* Manter registro de Compromissos e Agendamentos
* Autenticação local simples
